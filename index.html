Про мене

Я Сергій, інженер з комп’ютерних систем з великим досвідом у розробці та програмуванні мікроконтролерів, цифрових та аналогових схем, а також у створенні IoT пристроїв. Працюю з різними мовами програмування та інструментами розробки. Маю досвід у SMM, PR, написанні технічних статей та відеомонтажі.

  • Розробка та програмування плат на базі мікроконтролерів (AVR, ESP8266, ESP32, STM8, STM32)
  • Програмування додатків під Windows та Linux (Visual Studio, VSCode, PyCharm, Qt Creator)
  • Мови програмування: C, C++, JavaScript, Python
  • Моделювання цифрових та цифро-аналогових схем (Proteus 8)
  • GitHub / GitLab
  • SMM та PR
  • Монтаж/зйомка відео, фотозйомка
  • IoT (Internet of Things)
